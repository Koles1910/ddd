# KWA - Kosmiczni.pl Web Assistant

This is a browser extension bot for the game [kosmiczni.pl](https://kosmiczni.pl).

## Installation

To use this extension:

1. Download this repository.
2. Open your browser (Chrome/Edge/Brave).
3. Go to the Extensions page (`chrome://extensions/` or `edge://extensions/`).
4. Enable **Developer mode** (usually a toggle in the top right).
5. Click **Load unpacked**.
6. Select the `KWA` folder from this repository.

## Disclaimer

Use of bots in browser games may violate the game's terms of service. Use at your own risk.
